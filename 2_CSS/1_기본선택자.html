CSS(Cascading Style Sheets)

마크업 언어(html)가 실제로 화면상에 표시되는 방법을 기술하는 언어
W3C에서 CSS를 표준으로 지정

CSS 기본선택자

CSS 선택자의 종류

1. 모든 선택자(*)

모든 요소를 다 선택하고자 할 때 사용한다. 

        *{
            스타일 속성 : 값;
            스타일 속성 : 값;
            ...
        }

2. 태그 선택자

해당 문서 내에 같은 이름의 태그들을 모두 선택할 때 사용

        태그명, .... {
            속성 : 값;
        }

3. 클래스 선택자(.클래스명)

해당 문서 내에서 내가 원하는 "여러개의" 요소를 선택하고자 할 때 사용 

        .클래스명{
            속성 : 값;
        }

4. 아이디선택자(#아이디명)

해당 문서 내에 특정 html요소 "하나" 만을 선택하고자 할 때 사용 단, 해당 요소의 id 속성을 이용하여 고유한 아이디 값을 부여해야 한다. 

        
        #아이디명{
            ...
        }
